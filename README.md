# Лабораторная работа: Шифрование Эль-Гамаля (Вариант 2)

## Краткое описание

Программа реализует асимметричное шифрование Эль-Гамаля для побайтового шифрования файлов.

## Как использовать

### 1. Ввод параметров
- **p** — простое число (должно быть > 255). По умолчанию: `257`
- **x** — секретный ключ (диапазон: `1 < x < p-1`). По умолчанию: `5`
- **k** — случайное число (диапазон: `1 < k < p-1`, должно быть взаимно просто с `p-1`). По умолчанию: `7`

### 2. Получение генератора g
- Нажмите **«Рассчитать корни g»**
- Программа найдет все первообразные корни по модулю p
- Выберите один из корней в списке

### 3. Шифрование
- Нажмите **«Выбрать файл для шифрования»**
- Выберите любой файл
- Нажмите **«Зашифровать»**
- Результат: `encrypted_<имя_файла>.bin` (в той же папке)

### 4. Расшифрование
- Нажмите **«Расшифровать»**
- Выберите зашифрованный `.bin` файл
- Результат: `decrypted_<имя_файла>` (в той же папке)

## Формулы

- `a = g^k mod p`
- `b = (y^k * m) mod p`
